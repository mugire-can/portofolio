  ANIMATION KEYFRAMES ADDED (CSS)
───────────────────────────────────────────────────────────────────────────────

@keyframes fadeInUp
   Purpose: Fade in from bottom with slight movement
   Used by: Sections, skill cards, project cards
   Duration: 0.6s - 0.8s
   Effect: opacity 0→1, translateY 30px→0

@keyframes slideInLeft
   Purpose: Slide in from left side
   Used by: Certificate items
   Duration: 0.6s
   Effect: opacity 0→1, translateX -40px→0

@keyframes glowPulse
   Purpose: Pulsing cyan glow effect
   Used by: Skill cards on hover, project cards on hover
   Duration: 2s (repeating)
   Effect: box-shadow pulses in and out

@keyframes textGlow
   Purpose: Text shadow glow animation
   Used by: Section headings, form inputs on focus
   Duration: 3s (repeating)
   Effect: text-shadow intensity pulses

@keyframes floatHover
   Purpose: Up and down floating motion
   Used by: Buttons on hover
   Duration: 2s (repeating)
   Effect: translateY oscillates 0 to -8px

@keyframes ripple
   Purpose: Click ripple effect
   Used by: Button click effect
   Duration: 0.6s
   Effect: opacity 1→0

@keyframes shimmer (future use)

───────────────────────────────────────────────────────────────────────────────
  JAVASCRIPT FUNCTIONS ADDED
───────────────────────────────────────────────────────────────────────────────

function setupScrollAnimations()
   Purpose: Initialize Intersection Observer for scroll triggers
   Implementation:
   - Creates observer with 10% threshold
   - Monitors sections, cards, and items
   - Triggers fade-in animations when elements enter viewport
   - Unobserves after animation to save resources

function showToast(message, duration)
   Purpose: Display toast notifications instead of alerts
   Implementation:
   - Creates fixed position notification
   - Applies slideInLeft animation
   - Auto-dismisses after 3 seconds (customizable)
   - Uses CSS animation for smooth entry/exit
   - Styled with cyan borders and glow
