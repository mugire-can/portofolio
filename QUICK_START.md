## ğŸ”§ Want to Customize?

### Change animation speed:
Open `assets/css/my_portfolio.css`

Find:
```css
section {
    animation: fadeInUp 0.8s ease forwards;
}
```

Change `0.8s` to whatever you want:
- `0.5s` = faster
- `1.5s` = slower

### Disable a specific animation:
Find the animation in CSS and comment it out or delete it.

### Change glow colors:
Find hex colors like `#00d4ff` (cyan) and change to your color:
```css
@keyframes glowPulse {
    box-shadow: 0 0 25px rgba(0, 212, 255, 0.6);
    /* Change #00d4ff to any color you like */
}
```
